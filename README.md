# HotelRevAI – Revenue Analytics

HotelRevAI is a data analytics project focused on optimizing hotel revenue using Power BI. It analyzes booking data, customer behavior, pricing strategies, and occupancy trends to support data-driven decision-making.

## 📂 Project Modules

### Module 1: Data Preparation & Modeling
Creation of datasets and star schema design in Power BI.

### Module 2: KPI & Master Data
Development of key metrics such as ADR, RevPAR, and Occupancy%.

### Module 3: Data Transformation
Data merging, feature creation, and customer segmentation.

### Module 4: Cancellation & Demand Analysis
Analysis of cancellations, refunds, no-shows, and demand forecasting.

### Module 5: Revenue Dashboard
Interactive dashboard for revenue optimization and pricing strategy.

## 🛠 Tools Used
- Power BI  
- DAX  
- Excel  

## 🎯 Outcome
Provides insights to improve pricing, occupancy, and overall hotel revenue performance.
