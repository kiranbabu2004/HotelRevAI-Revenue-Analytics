🏨 HotelRevAI – Revenue Analytics Dashboard
📌 Project Overview

HotelRevAI is a data analytics project designed to analyze hotel performance using Power BI. The project focuses on understanding booking trends, revenue generation, and operational efficiency through interactive dashboards.

It enables hotel management to make data-driven decisions using key performance indicators and visual insights.

🛠️ Tools & Technologies Used
Power BI
Microsoft Excel
DAX (Data Analysis Expressions)
📊 Dataset Description

The dataset includes hotel booking and guest-related information such as:

Booking details
Customer information
Room types
Revenue data
Dates and duration

This data is structured and modeled for efficient analysis.

📈 Key Performance Indicators (KPIs)
ADR (Average Daily Rate)
→ Average revenue earned per occupied room
RevPAR (Revenue Per Available Room)
→ Measures overall revenue performance
Occupancy Rate
→ Percentage of occupied rooms
👨‍💼 GM Dashboard (General Manager View)

The GM Dashboard provides a high-level overview of hotel performance:

Total revenue trends
Booking patterns over time
Occupancy insights
Overall business performance

👉 Helps in strategic decision-making

📊 RM Dashboard (Revenue Manager View)

The RM Dashboard focuses on revenue optimization:

Revenue by booking source (OTA, Direct, etc.)
ADR and RevPAR analysis
Pricing and demand trends

👉 Helps in maximizing revenue and pricing strategy

🧩 Data Model (Star Schema)

The project follows a Star Schema design:

Central Fact Table (Bookings data)
Connected Dimension Tables (Date, Customer, Room Type, etc.)

This improves:

Query performance
Data clarity
Analytical efficiency

✅ Conclusion

HotelRevAI provides meaningful insights into hotel operations by analyzing key metrics and trends. The dashboards help management:

Improve revenue strategies
Understand customer behavior
Optimize occupancy and pricing

This project demonstrates how data analytics can enhance decision-making in the hospitality industry
